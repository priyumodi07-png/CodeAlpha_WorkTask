## 📌 Project: CodeAlpha-chatbot

### Overview
The **Chatbot** project is a simple Python program designed to respond to user queries interactively.  
It demonstrates the use of conditional logic, loops, and input/output handling to simulate a conversational agent.

---

### Features
- Interactive console-based chatbot.
- Responds to user input with predefined answers.
- Demonstrates basic natural language handling using conditionals.
- Simple and beginner-friendly design.

---

### Key Concepts Used
- **[Conditional Statements](ca://s?q=Python_conditional_statements)** → to determine chatbot responses.  
- **[Loops](ca://s?q=Python_loops_usage)** → to keep the conversation running until the user exits.  
- **[Input/Output](ca://s?q=Python_input_output_in_Python)** → to take user queries and display responses.  
- **[Functions](ca://s?q=Python_functions_basics)** → optional modular design for cleaner code.
